# Finance Projects

A portfolio of finance projects built to develop and demonstrate skills across four areas of finance, aimed at internship preparation.

## Structure

- [`IB/`](IB/) — Investment Banking: valuation, M&A, comps, LBOs, pitch books
- [`PE/`](PE/) — Private Equity: LBO modeling, fund mechanics, deal screening
- [`VC/`](VC/) — Venture Capital: cap tables, startup valuation, fund modeling
- [`quant/`](quant/) — Quantitative Finance: backtesting, portfolio optimization, risk modeling

Each folder contains a `README.md` with a project roadmap (basic → complex) and subfolders for individual projects as they're built.
