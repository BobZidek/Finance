# Venture Capital Projects

## Basic
1. **Cap table and dilution tracker** across funding rounds (seed → Series C)
2. **SaaS metrics dashboard**: CAC, LTV, churn, burn multiple for a sample startup
3. **Simple startup valuation** using the VC method (target exit value / required return)

## Medium
4. **Term sheet analyzer**: liquidation preferences, participating vs. non-participating, anti-dilution across exit scenarios
5. **Portfolio construction model**: power-law return modeling across 20 hypothetical investments

## Complex
6. **Fund model**: reserves strategy, follow-on decisions, fund-level DPI/TVPI projections
7. **Market-sizing + competitive landscape deep dive** (TAM/SAM/SOM) for an emerging sector (e.g. AI infra) with investment thesis
8. **Full investment memo**: thesis, market, team, model, risks, recommendation — real or realistic mock startup
