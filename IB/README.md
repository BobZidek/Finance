# Investment Banking Projects

Sectors are varied deliberately across projects so the portfolio doesn't read as single-industry.

## Basic
1. **Comp analysis — fast food chains** (MCD, YUM, QSR, WEN, SHAK): EV/EBITDA, P/E, margins
2. **DCF model — semiconductors** (e.g. Texas Instruments): sensitivity table on WACC/terminal growth
3. **M&A accretion/dilution calculator — regional banks**: simple stock+cash deal, toggle inputs

## Medium
4. **Comps + football field valuation — athletic apparel** (NKE, LULU, UAA, DECK): trading comps, precedent transactions, DCF combined
5. **LBO model — healthcare services** (dental/vet roll-up chain): debt schedule, IRR/MOIC sensitivity

## Complex
6. **Merger model with synergies — airlines**: financing structure, pro forma EPS impact, back-tested against a real historical deal
7. **Multi-sector comps dashboard — software/SaaS**: auto-pulls financials via API, computes multiples, flags over/undervalued names
8. **Full pitch book — industrials/logistics**: industry overview, comps, DCF, LBO, precedent transactions, recommendation
