# Potential Enhancements

- **Dis-synergy adjustment**: net the SOTP upside against an estimated value of shared
  services, cross-selling, and procurement scale that would be lost upon separation, for a
  more complete "net of separation costs" conclusion.
- **Standalone cost build-up**: model the incremental standalone public-company costs each
  spun-off segment would incur (new finance/legal/IR functions), which real spin-off analyses
  must account for explicitly.
- **Tax treatment of the separation**: model whether the separation would be tax-free (e.g., a
  qualifying Reverse Morris Trust or spin-off structure) vs. taxable, since a taxable
  transaction could destroy much of the modeled value creation via transaction taxes.
- **DCF cross-check per segment**: build a standalone DCF for each segment (reusing the DCF
  engine from [`IB/dcf-model-semiconductors`](../dcf-model-semiconductors)) as a valuation
  cross-check alongside the segment trading multiples.
- **Precedent spin-off performance**: add a section referencing how comparable historical
  spin-offs performed post-separation as a real-world sanity check on the thesis.
- **Multiple scenario segment growth**: extend beyond a static current-year EBITDA snapshot to
  a forward-looking segment-level growth projection, since SOTP theses are often built on
  forward (not trailing) financials.
- **Sensitivity by individual segment**: extend the sensitivity table to vary each segment's
  multiple independently (a 3D or tornado-chart sensitivity) rather than moving all three
  together, to identify which segment's multiple assumption the conclusion is most sensitive to.
