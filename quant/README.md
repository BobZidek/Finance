# Quantitative Finance Projects

## Basic
1. **Moving-average crossover backtest** on historical price data
2. **Portfolio optimizer**: mean-variance (Markowitz) efficient frontier for 5-10 stocks
3. **Options pricing calculator** (Black-Scholes) with Greeks

## Medium
4. **Factor model**: build and test a multi-factor (value/momentum/quality) stock ranking system
5. **Pairs trading / statistical arbitrage** with cointegration testing

## Complex
6. **Risk model**: VaR/CVaR estimation via historical, parametric, and Monte Carlo methods for a portfolio
7. **ML return-prediction model**: feature engineering from price/fundamental data, backtested with realistic transaction costs
8. **Full systematic strategy**: signal generation, portfolio construction, risk management, backtesting engine with performance attribution
