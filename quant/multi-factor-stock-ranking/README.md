# Multi-Factor Stock Ranking System (Value / Momentum / Quality)

Computes standardized Value, Momentum, and Quality factor scores for 80 synthetic stocks,
combines them into a composite score, ranks into quintiles, and evaluates the ranking against
realized forward returns via the Information Coefficient — the standard factor research metric.

For the theory behind each factor and an honest note on the synthetic signal-to-noise ratio,
see [`theory/factor_investing_theory.md`](theory/factor_investing_theory.md).

## What the code does

1. **`generate_data.py`**: generates 80 synthetic stocks with P/E ratio, 12-month trailing
   return, and ROE, plus a forward 1-month return **deliberately generated as a noisy function
   of each stock's true factor exposures** — so there's a genuine (if noisy) relationship for
   the backtest to recover, saved to `data/stock_factors.csv`.
2. **`factor_model.py`**:
   - Converts P/E to earnings yield and computes z-scores for Value, Momentum, and Quality.
   - Combines the three into an equal-weighted Composite score.
   - Assigns stocks to quintiles by Composite score.
   - Computes the **Information Coefficient** (Spearman rank correlation with forward return)
     for each individual factor and the Composite.
   - Computes average forward return by quintile and the long-short (Q5 − Q1) spread.
   - Outputs the ranked stock table, IC table, quintile returns table, and three charts.

## Data note

Live fundamental/price data wasn't reachable in this environment (network/SSL restriction),
so this project uses a **reproducible synthetic dataset** (fixed random seed) with a
deliberately generous signal-to-noise ratio to clearly demonstrate the methodology — see the
theory doc for why real single-period factor signals are much weaker and noisier than this.

## How to run

```bash
cd code
pip install -r requirements.txt
python generate_data.py
python factor_model.py
```

Outputs are written to `output/`.

## Key results (from the included synthetic data)

- **All three factors show positive, statistically significant Information Coefficients**:
  Value 0.38 (p=0.0005), Momentum 0.29 (p=0.0096), Quality 0.22 (p=0.054).
- **The Composite score's IC (0.51) exceeds every individual factor** — combining independent
  signals averages out idiosyncratic noise while preserving genuine predictive content, the
  same diversification logic behind portfolio construction itself.
- **Average forward return rises monotonically from Q1 (-6.4%) to Q5 (+3.7%)** — the clean
  "staircase" pattern factor researchers look for as visual confirmation of a genuine ranking.
- **Long-short (Q5 − Q1) spread: 10.14%** for the 1-month period tested.
