# Full Systematic Strategy — Signal, Portfolio Construction, Risk Management & Attribution

The quant capstone project: a complete systematic equity strategy pipeline over a 40-stock,
36-month panel — Value/Momentum/Quality signal generation, dollar-neutral long/short portfolio
construction, volatility-targeted risk management, turnover-based transaction costs, and
regression-based performance attribution back to the underlying factors.

For the theory behind each piece, and **an honest account of a real recalibration this
project's own development required**, see
[`theory/systematic_strategy_theory.md`](theory/systematic_strategy_theory.md).

## What the code does

1. **`generate_data.py`**: generates a 40-stock x 36-month panel of Value/Momentum/Quality
   factor scores and forward monthly returns (each month drawn independently), with a known,
   injected factor relationship — carefully calibrated for portfolio-level (not just
   individual-stock) signal strength (see the theory doc for why this required a real fix).
2. **`systematic_strategy.py`**:
   - **Signal generation**: builds the equal-weighted Composite score each month, extending
     [`quant/multi-factor-stock-ranking`](../multi-factor-stock-ranking) into a full panel.
   - **Portfolio construction**: dollar-neutral long top-quintile / short bottom-quintile,
     rebuilt every month.
   - **Risk management**: scales gross exposure via trailing 6-month realized volatility
     targeting (8% annualized target, capped at 2x leverage).
   - **Transaction costs**: turnover-based cost (15bps) applied at every rebalance.
   - **Performance attribution**: regresses net strategy returns on each factor's own monthly
     long/short return to decompose where performance actually came from.
   - Outputs the full monthly backtest detail, a performance summary (net/gross/unscaled), an
     attribution table, an equity curve chart, an attribution bar chart, and a
     signal-vs-vol-scalar chart.

## Data note

Live market data wasn't reachable in this environment (network/SSL restriction), so this
project uses a **reproducible synthetic panel** (fixed random seed), deliberately recalibrated
after an initial unrealistic result (see the theory doc) to produce believable performance
statistics.

## How to run

```bash
cd code
pip install -r requirements.txt
python generate_data.py
python systematic_strategy.py
```

Outputs are written to `output/`.

## Key results (from the included synthetic data)

- **Net of costs**: 36.8% total return over 3 years (10.9% annualized), **9.1% realized
  volatility (close to the 8% target)**, **0.75 Sharpe ratio**, -9.8% max drawdown.
- **Transaction costs consumed 8.45 percentage points** of return — gross Sharpe was 1.07,
  falling to 0.75 net, a material and honestly-reported cost drag.
- **Performance attribution (R² = 0.756)**: all three factors show positive, statistically
  significant contributions (Value beta 0.29, Momentum 0.44, Quality 0.58, all p < 0.01) —
  meaning the strategy's returns are genuinely explained by its stated factor exposures, not
  unexplained noise.
- An earlier version of this project produced an obviously unrealistic Sharpe above 9 with
  zero drawdown — caught, diagnosed (portfolio-level averaging amplifies individual-stock
  signal strength), and fixed by recalibrating the injected signal — documented in the theory
  doc rather than silently corrected.
