# Moving-Average Crossover Backtest

Backtests a classic 50/200-day SMA crossover ("golden cross") trend-following strategy against
buy-and-hold on a synthetic 3-year daily price series spanning three distinct regimes
(uptrend, choppy/sideways, strong uptrend).

For the theory behind the strategy and an honest read of why it underperformed here, see
[`theory/moving_average_crossover.md`](theory/moving_average_crossover.md) — **this project
deliberately reports a losing backtest and explains why, rather than only showing a winning one.**

## What the code does

1. **`generate_data.py`**: generates a reproducible synthetic 756-day (3-year) daily price
   series via geometric Brownian motion, with three regime-specific drift/volatility settings
   (uptrend, choppy/sideways, strong uptrend), saved to `data/synthetic_prices.csv`.
2. **`backtest.py`**:
   - Computes 50-day and 200-day simple moving averages and the resulting long/flat signal.
   - **Shifts the signal forward one day** before applying it to returns, avoiding look-ahead bias.
   - Computes strategy and buy-and-hold equity curves, CAGR, annualized volatility, Sharpe
     ratio, and max drawdown.
   - Breaks down performance by regime.
   - Outputs an equity curve chart and a price/signals chart with buy/sell markers.

## Data note

Live market data wasn't reachable in this environment (network/SSL restriction), so this
project uses a **clearly-labeled, reproducible synthetic price series** (fixed random seed)
rather than real historical prices. The backtest mechanics — signal generation, look-ahead-bias
avoidance, performance measurement — are exactly what you'd run against real data; swap
`data/synthetic_prices.csv` for a real OHLCV dataset to make this an actual historical backtest.

## How to run

```bash
cd code
pip install -r requirements.txt
python generate_data.py
python backtest.py
```

Outputs are written to `output/`.

## Key results (from the included synthetic data)

- **The strategy underperformed buy-and-hold**: 12.5% total return / 4.0% CAGR / 0.07 Sharpe,
  vs. buy-and-hold's 25.0% total return / 7.7% CAGR / 0.28 Sharpe.
- **The strategy underperformed in every single regime tested**, including losing money
  (-2.7%) during the "Strong Uptrend" regime where buy-and-hold gained 4.3%.
- Only **3 trades** occurred across the full 3-year backtest — the 200-day SMA's warmup period
  meant the strategy couldn't take any position at all for the first ~200 days, missing a
  meaningful chunk of the initial uptrend regime by construction, independent of whether the
  underlying trading rule itself was sound.
- Max drawdown was only marginally better than buy-and-hold (-21.1% vs. -22.9%) — the
  crossover's inherent lag meant it didn't avoid the deepest decline either.
