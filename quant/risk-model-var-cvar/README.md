# Portfolio Risk Model — VaR / CVaR via Three Methods

Estimates 1-day Value-at-Risk and Conditional VaR for a 5-asset, $10mm portfolio three ways —
historical simulation, parametric (normal distribution), and Monte Carlo (correctly-specified
fat-tailed Student-t) — deliberately built to show how the normal-distribution assumption
understates real tail risk, and validates the finding via VaR backtesting.

For the theory behind each method and why the normal assumption fails at high confidence
levels, see [`theory/var_cvar_theory.md`](theory/var_cvar_theory.md).

## What the code does

1. **`generate_data.py`**: generates 3 years of daily returns for a 5-asset portfolio using a
   **correctly-constructed multivariate Student-t distribution** (shared chi-square scaling
   factor across assets each day) with a hand-specified correlation matrix — genuinely
   fat-tailed, unlike a naive blend of normal and t-distributed draws.
2. **`risk_model.py`**:
   - Computes **Historical** VaR/CVaR directly from the empirical return distribution.
   - Computes **Parametric (Normal)** VaR/CVaR from closed-form normal distribution formulas.
   - Computes **Monte Carlo** VaR/CVaR by simulating 50,000 days from a correctly-specified
     multivariate Student-t model (matching the true fat-tailed data-generating process).
   - **Backtests** each method's VaR by counting actual historical exceedances against the
     number expected from the stated confidence level.
   - Outputs a comparison table, a return-distribution chart (historical vs. Monte Carlo vs.
     the parametric normal fit), and a VaR-by-method bar chart.

## Data note

Live market data wasn't reachable in this environment (network/SSL restriction), so this
project uses a **reproducible synthetic, deliberately fat-tailed dataset** (fixed random
seed) — real asset returns are also empirically fat-tailed, which is exactly why this
comparison matters for real portfolios, not just this synthetic one.

## How to run

```bash
cd code
pip install -r requirements.txt
python generate_data.py
python risk_model.py
```

Outputs are written to `output/`.

## Key results (from the included synthetic data)

At **99% confidence**, on a $10mm portfolio:

| Method | 1-Day VaR | Backtested Exceedances (expected: 7.56) |
|---|---|---|
| Historical | $159,292 | 8 |
| Parametric (Normal) | $126,350 | **14 — nearly 2x expected** |
| Monte Carlo (Student-t) | $143,836 | 11 |

- **Parametric (Normal) VaR understates historical VaR by ~21%** at the 99% level, and the
  backtest confirms the consequence directly: it was breached **nearly twice as often as its
  own stated confidence level implies**.
- **Monte Carlo using the correctly-specified fat-tailed model tracks the historical result
  much more closely** — the problem is specifically the normal-distribution assumption, not
  simulation-based VaR in general.
- **The gap between methods is much smaller at 95% confidence** than at 99% — fat tails matter
  far more the deeper into the distribution's tail you look.
